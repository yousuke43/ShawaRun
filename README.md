# ShawaRun (シャワラン！)
#### 🚿 シャワーを浴びて、一日の行動スイッチをONに！

## 1. 概要 (Introduction)

本システムは、朝の行動を促進する習慣化支援アプリです。朝、シャワーや入浴が億劫で、その日一日をだらだらと過ごしてしまうというユーザーに対し、シャワーを「行動のトリガー」として活用することで、生産性の高い一日をスタートするための支援を提供します。

## 2. 主要機能 (Features)

本アプリの中核をなす機能群です。

### 2.1. シャワー記録とトリガー機能
* シャワー・入浴の開始時刻と完了時刻を記録します。
* 完了時に「行動スイッチON」のメッセージと、タスク記入画面へ自動で遷移させます。

### 2.2. Todo リストと行動促進
* **Todo 記入:** シャワー完了直後に、その日の行動計画（タスク）を記入します。
* **達成管理 (✅):** 記入したタスクの完了チェック機能を提供します。

### 2.3. 感想とフィードバック
* **感想記入:** シャワー完了後、簡単な感想（「きもちいい！」「入ってよかった」など）を記録します。
* **振り返り:** 過去のポジティブな感想をランダム表示し、次の入浴へのモチベーションを高めます。

### 2.3. 通知機能
* **通知:** 時間を入力し、シャワー入って！という通知が来るようにできる。アラームみたいなイメージで湖底ではなく、好きな時間に設定できるように


## 3. アプリの設計指針とデータ活用

### 3.1. 履歴データの保存
* **対象データ:** シャワー時刻、Todoリスト（記入内容と達成状況）、感想を日次で保存します。
* **目的:** ユーザーの行動の変化（行動力・達成率）を可視化し、成長実感を提供するため。

### 3.2. フィードバックと相関分析 (重要)
* 記録された感想のポジティブ度と、その日のTodo達成率を比較できるダッシュボードを設計します。
* これにより、「シャワーの効果」をデータに基づきユーザーに示し、習慣化を強化します。
